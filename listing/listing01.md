# listing01

Что выведет программа? Объяснить вывод программы.

```go
package main
 
import (
    "fmt"
)
 
func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:

```
Вывод: [77, 78, 79]
Так как берется срез от слайса a, то и значения будут от первого, до 4
(если говорить интервалами, как в математике, то [1, 4))
```
