# listing02

Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и порядок их вызовов.

```go
package main
 
import (
    "fmt"
)
 
func test() (x int) {
    defer func() {
        x++
    }()
    x = 1
    return
}
 
 
func anotherTest() int {
    var x int
    defer func() {
        x++
    }()
    x = 1
    return x
}
 
 
func main() {
    fmt.Println(test())
    fmt.Println(anotherTest())
}
```

Ответ:

```
Вывод:
2
1

Почему?
В функции test выходным значением является х, а defer вызывается в конце функции,
следовательно когда мы инкрименируем в лямбда функции x, то инкрименируем
мы выходное значение

В функции anotherTest выходным значением является любое значение типа int
(то, которое мы передадим через return) и на момент выполнения defer'а
возвращаемое значение и переменная x – разные вещи

По поводу defer

Все функции, которые мы передадим в defer будут выполняться после выполнения
функции, где эти defer'ы и были объявлены. Но выполняться они будут
по очереди добавления в стек вызовов (чем раньше defer, тем позже он выполнится)
```
